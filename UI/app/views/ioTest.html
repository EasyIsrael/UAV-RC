Socket.io Test

Socket.io is {{{false:'connecting',true:'connected'}[socket.connected]}}

The purpose of this page is to test the socket.io network for three scenarios:

  • sending/receiving a message to/from a JavaScript-module
  • sending/receiving a message to/from a Python-module
  • sending/receiving a message to/from itself (the controller of this page; taking a roundabout route over the server)

Clicking on "send" emits a message to the corresponding module via socket.io. The message has a date property that holds 3 timestamps (when the message was "send", has "arrived" at the module and was "recieved" back at the controller of this page). The message also has a "count" property that is incremented by the module (as a proxy for real work).
Checking the "repeat" checkbox will refire a message to the module after receiving a answer from the module (uncheck it to stop the loop).
